===== 컴퓨터에서 파일 열기: npm =====
컴퓨터에서 로컬에서 전자카탈로그를 열려면 CORS 보안 제한으로 인해 웹 서버가 필요합니다.
파일:// 프로토콜.npm을 사용하여 "http-server"를 설치하고 실행할 수 있습니다.

1.맥에서 홈브루를 통해 npm 설치하기 - https://butter-ring.tistory.com/17
2.윈도우에서 npm 설치하기 - https://blog.naver.com/jonghong0316/221917408861
3.이 주소에서 열기 - http://127.0.0.1:8080


===== 웹사이트에 업로드하기 =====
1.zip 파일 압축을 풀고, 내부의 압축파일도 풀어줍니다.
2.웹서버에 해당 폴더를 올려줍니다.
3.즉시 작동해야 하지만 index.html의 URL이 웹 사이트의 정확한 상대 URL을 참조하는지 확인하십시오. 
다른 파일을 변경할 필요가 없습니다.
